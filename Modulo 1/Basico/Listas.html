Para adicionar um item numa lista usasse a Tag li (List item)

Pode-se alterar o tipo da ordenação com :

( type A ou a / Para ordem feita com letras )
( type I e i / Para numeros romanos )
(type 1 / Para ordem numerica )

OBS: Pode-se tambem mudar o incio com a tag start e o numero ou letra que se quer começar
